# EtherX Standalone Browser

**Complete standalone version** — Electron desktop app + web version.

## 🚀 Downloads

Visit [GitHub Releases](https://github.com/ktrucek/etherx-standalone/releases) for:
- 🐧 **Linux**: AppImage + .deb
- 🪟 **Windows**: Portable .exe + ZIP
- 🍎 **macOS**: DMG (Intel + Apple Silicon)
- 🌐 **Web**: index.html (no installation)

## 📦 Quick Start

### Desktop App
Download from releases and run. No installation needed for portable versions.

### Web Version
Open `src/index.html` in any modern browser.

## 🔨 Build from Source

```bash
npm install
npm start          # Run dev
npm run dist:linux # Build Linux
npm run dist:win   # Build Windows
npm run dist:mac   # Build macOS
```
