🌐 CORS Napomena (Standalone Web Verzija)
CORS (Cross-Origin Resource Sharing) je sigurnosna politika browsera koja blokira web stranicama pristup resursima s drugih domena.

Standalone web verzija EtherX browsera nema pristup CORS proxy-ju koji zaobilazi ovu politiku. Zato neke funkcije (npr. DevTools Source viewer, CSP header fetch) mogu biti ograničene.

Rješenje: Koristi Electron desktop verziju za puni pristup svim funkcijama ili implementiraj vlastiti CORS proxy (Service Worker).

📷 Screenshot funkcija

Snimanje zaslona možete pokrenuti na tri načina:

  • Izbornik View → Take Screenshot — snima vidljivi dio stranice
  • Izbornik View → Full Page Screenshot — pokušava snimiti cijelu stranicu
  • Tipkovnički prečac — zadano Ctrl+Shift+S, može se promijeniti u Settings → Advanced → Screenshots

Lokacija spremanja: postavite u Settings → Advanced → Save screenshots to. Opcije: automatsko preuzimanje (PNG), kopiranje u međuspremnik ili odabir svaki put.

🔧 Prilagodba alatne trake
